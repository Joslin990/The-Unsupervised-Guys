# The Unsupervised Guys

A static promotional website for the gaming brand "The Unsupervised Guys." The site highlights the channel, featured videos, social links, contact details, and a merch shop page.

## Project Overview

- `index.html` - Main landing page with home, about, videos, contact sections, and a navigation menu.
- `main.html` - Redirect page that forwards visitors to `index.html`.
- `shop.html` - Merch shop page showcasing product listings and pricing.
- `style.css` - Global styling for page layout, typography, and responsive visuals.
- `script.js` - Client-side script included by the pages.
- `images/` - Image assets used for banners, logos, icons, and social media visuals.

## Features

- Full static website with no backend required.
- Embedded YouTube videos for channel content.
- Social links to TikTok, Instagram, Reddit, and YouTube.
- Merch shop page with product cards and pricing.
- Custom font loading from Google Fonts.
- Favicon support for browser tabs.

## Getting Started

1. Clone or download the project files.
2. Open `index.html` in a browser.
3. Browse the home page, videos section, contact links, and shop page.

## Notes

- `main.html` is configured to immediately redirect visitors to `index.html`.
- The site is designed to be served as a static website from any web server or local file system.
